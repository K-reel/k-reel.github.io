<h3><a href="https://github.com/K-reel/artifactor">Artifactor</a></h3>

A Python CLI tool for generating Jekyll HTML posts from structured article data. Treats writing like an artifact pipeline with deterministic, reproducible output.

**Key Features:**
- **Offline-first testing** with fixtures — develop and test without external dependencies
- **Adapter pattern** for multiple content sources — easily extend to support new input formats
- **Deterministic output** — same input always produces identical bytes
- **CI/CD ready** — designed for GitHub Actions and automated publishing workflows

**Tech Stack:** Python, Jekyll, GitHub Actions

**Status:** Active development
