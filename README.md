# k-reel.github.io

Personal portfolio site hosted at [https://k-reel.github.io/](https://k-reel.github.io/)

## Structure

- Root site: Portfolio and project showcase
- `/artifactor/`: [Artifactor demo site](https://k-reel.github.io/artifactor/) (published from [K-reel/artifactor](https://github.com/K-reel/artifactor))

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit http://localhost:4000

## Deployment

GitHub Pages deploys automatically from the `main` branch (root).
